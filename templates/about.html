About

This is an English to Kannada dictionary, powered by Alar Data. Alar is an authoritative Kannada-English dictionary corpus created by V. Krishna. It contains over 150,000 Kannada words with over 240,000 English definitions. The Alar dictionary was released by Zerodha team, and is available here. Read the full story here. This dictionary is created by using The alar dataset, for the lookup of English to Kannada words we use word embeddings.

Data

The entire corpus (© V. Krishna) is available on the Alar repository. It is licensed under ODC-ODbL.

Website

This website is published using dictmaker, and uses knphone, a Kannada phonetic indexing algorithm, for search. Search suggestions and transliterations are powered by Varnam.

This is a work in progress project, free for use for everyone. Technical feedback can be sent to sumitj39@outlook.com.
